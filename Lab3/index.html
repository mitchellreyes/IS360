Inheritance

Inheritance is the process by which properties are passed from parent to child elements even though those properties have not been explicitly defined by other means. Certain properties are inherited automatically, and as the name implies, a child element will take on the characteristics of its parent with regards to these properties.

Note that modern browsers operating in quirks mode behave a little buggily in that they don’t inherit certain properties into tables. This quirks mode behavior emulates the buggy behavior of much older browsers. Usually, these properties have to be specifically applied to the table element, although a better approach would be to avoid the issue altogether by using a doctype that causes the browser to use standards mode rendering.

For example, if a div element has a font-size of 20px then, assuming that no other font-size declarations have been explicitly defined, any children will also inherit that font-size value. So there is no need to change it.
